# 日记本小程序

微信小程序日记应用，用于记录日常心情、事件和习惯。基于微信云开发，数据存储在云端，无需额外后端。

## 功能

- **时光页**：月历视图 + 心情色点 + 近期记录流
- **记一笔**：选择心情、标签（支持自定义参数）、文字记录
- **日视图**：查看某天所有记录，支持编辑和删除
- **标签系统**：预设标签 + 自定义标签，支持文本/数值参数
- **数据导出**：JSON 完整备份 / CSV 表格格式
- **深色模式**：跟随系统自动切换

## 视觉风格

宣纸 + 淡墨 + 朱砂印章

## 技术栈

- 微信小程序原生开发
- 微信云开发（CloudBase）
- 自定义导航栏 + 自定义 TabBar

## 项目结构

```
miniprogram/
├── components/nav-bar/    # 自定义导航栏组件
├── custom-tab-bar/        # 自定义底部导航
├── pages/
│   ├── timeline/          # 时光页（首页）
│   ├── record/            # 记一笔
│   ├── day/               # 日视图
│   ├── mine/              # 我的
│   ├── insight/           # 规律（待开发）
│   ├── tag-manage/        # 标签管理
│   └── tag-create/        # 新建标签
└── utils/
    ├── config.js          # 配置常量
    ├── db.js              # 数据库操作
    └── util.js            # 工具函数
```

## 云数据库

| 集合 | 说明 | 权限 |
|------|------|------|
| `tags` | 标签定义 | 仅创建者可读写 |
| `fragments` | 日记片段 | 仅创建者可读写 |

## 开发

1. 使用微信开发者工具打开项目
2. 在云开发控制台创建 `tags` 和 `fragments` 集合
3. 集合权限设为「仅创建者可读写」
4. 编译运行
